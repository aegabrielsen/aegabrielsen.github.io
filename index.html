Park Finder

JavaScript, MongoDB, Express, EJS, NodeJS, AJAX, RESTful API

Full-stack CRUD app that allows users to share and rate their favorite parks. Made entirely with JavaScript and NodeJS. This project is hosted online and includes security and geo services using Mapbox. User credentials are hashed and stored safely in a MongoDB NoSQL database.

Dynamic Allocator

C, Linux, Emacs

Dynamic memory allocator that I created for a systems programming class. This process replaces the default malloc, calloc, realloc, and free functions for C programs on POSIX X86-64. It requests memory directly from the system and stores it on a table to serve out to programs. Bit manipulation is used to store extra information inside 8 byte metadata to save space. Can run both default Unix commands like ls and echo along with more advanced programs like Vim. A lot of students dreaded this project but I thought it was a really interesting challenge. Photo is an unrelated picture of a clone of Conway's Game of Life that I also made in C.
